babylon.js x TypeScript My Practice

# Reference page
[Babylon.js First Steps](https://doc.babylonjs.com/babylon101/first)

## Build Setup

``` bash
# install dependencies
$ npm i

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build

# Lint Fix
$ npm run lint

```
